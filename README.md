# Subtitle Creator

Generate synced subtitles for any video, in English or Hebrew, automatically.

Drop in a video, pick a language, get an SRT file. No timing, no editing.

- English subtitles: transcribes speech from any language into English
- Hebrew subtitles: transcribes and translates to Hebrew, right-to-left

## Requirements

- Windows 10 or 11 (64-bit)
- [FFmpeg](https://ffmpeg.org/download.html) on PATH
- Python 3.10+
- Internet connection

## Setup

1. Download and extract the zip
2. Run `scripts\setup.ps1` once to install dependencies
3. Open `SubtitleCreator.exe`

## How to use

1. Drop a video file or click Browse
2. Click English Subtitles or Hebrew Subtitles
3. Wait for it to finish — you can see progress in real time
4. The SRT file gets saved next to your video, or in the folder you pick in Settings

## Settings

You can change the theme (light/dark/system) and choose where subtitle files get saved.
